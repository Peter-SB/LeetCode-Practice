Just count the number of 1s in the string representaion 

### Hacky 
```python
class Solution:
    def hammingWeight(self, n: int) -> int:
        return str(bin(n))[2:].count("1")
```

### Using Bitwise
```python
    def hammingWeight(self, n: int) -> int:
        ones = 0

        while n > 0:
            if n%2:
                ones += 1
            n >>= 1

        return ones
```
